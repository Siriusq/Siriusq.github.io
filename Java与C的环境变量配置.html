Java与C的环境变量配置

发表于 2020-08-18 分类于 Tech 阅读次数：
本文字数： 661 阅读时长 ≈ 1 分钟
昨天又把系统重装了，升级到了2004，本以为能凭借记忆配置好Java和C的环境变量，然鹅最后还是找了百度，所以就再记录一遍吧

Java环境变量配置

  1. 直接在任务栏搜索框搜索环境变量，会出现编辑系统环境变量
  2. 在弹出的窗口中点击右下角的环境变量
  3. 在下面的系统变量中新建系统变量，变量名JAVA_HOME，变量值为JDK的安装路径，默认是C:\Program Files\Java\jdk-14.0.2
  4. 再新建一个系统变量CLASSPATH，变量值为.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;
  5. 双击系统变量中的path，点击右上角的新建
  6. 分别加入%JAVA_HOME%\bin与%JAVA_HOME%\jre\bin
  7. 确定三连回到桌面
  8. 打开终端，输入java -version检查是否成功，成功的话会显示java版本号
  9. 不成功的话重启一遍，并检查一下变量是否输入正确

C环境变量配置

  1. 在mingw中勾选mingw32-base和mingw-gcc-g++就行，选择多了会装到天荒地老。。。勾选方式是右键 - Mark for Installation
  2. 点击右上角的Installation - Apply Changes
  3. 在弹出的窗口中点击Apply确认
  4. 漫长的等待。。。。。。土豆服务器经常会报错安装失败
  5. 安装完成后同上打开系统变量
  6. 双击系统变量中的path，点击右上角的新建
  7. 加入mingw的安装路径，默认是C:\MinGW\bin
  8. 打开终端，输入gcc -v检查是否成功
  9. 不成功同样重启试试

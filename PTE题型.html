PTE题型

发表于 2019-05-11 更新于 2022-05-23 分类于 留学 本文字数： 1.2k 阅读时长 ≈ 5 分钟
被雅思恶心到了（~~口语万年5.5~~），转战PTE，开个坑记录下PTE的题型，文章的题型顺序是按照考试的时间先后顺序排列的。另外[送上PTE官网](https://pearsonpte.com/)，可以在官[这里](https://pearsonpte.com/the-test/who-accepts-it/)查询下自己想申请的学校接不接受PTE成绩，另外也可以到学校官网查询，例如我要申请的UCD是要达到总分63小分不低于59，对应雅思就是总分6.5小分6.0的水平，但是PTE是全机器判卷，排除掉了人为因素的影响，更为客观，AI才是未来~~

口语

口语部分共有5个题型

Read Aloud

  • 三至四行的短文本，朗读出来
  • 40s准备时间
  • 40s朗读时间

评分标准

  • 内容
  • 发音
  • 流利度

交叉评分

  • 阅读
  • 口语

Repeat Sentence

  • 听录音
  • 马上复述
  • 包含10-12个单词

评分标准

  • 内容3分
  • 发音5分
  • 流利度5分

交叉评分

  • 听力
  • 口语

Discribe Image

  • 给出一个图表
  • 40s准备
  • 描述图表

评分标准

  • 内容5分
  • 发音5分
  • 流利度5分

交叉评分

  • 口语

Retell Lecture

  • 播放一段Lecture录音
  • 可能有背景音
  • 口音多样
  • 10s准备
  • 40s概括复述

评分标准

  • 内容5分
  • 发音5分
  • 流利度5分

交叉评分

  • 口语
  • 听力

Answer Short Question

  • 一个小问题
  • 用几个单词简短回答

评分标准
答对1题1分

交叉评分

  • 口语
  • 听力

写作

写作部分共有2个题型

Summarize Written Text

  • 三至四道题
  • 每个题三至四段的文本
  • 总结主旨
  • 1句话概括
  • 5-75个单词

评分标准

  • 内容2分
  • 形式1分
  • 语法2分
  • 词汇2分

交叉评分

  • 阅读
  • 写作

Write Essay

  • 大作文
  • 一般是1篇，非酋可能会抽到2篇（瑟瑟发抖）
  • 马上复述
  • 200-300词

评分标准

  • 内容3分
  • 形式2分
  • 结构及连贯性2分
  • 语法2分
  • 语言范围（是否多样）5分
  • 词汇2分
  • 拼写2分

交叉评分

  • 写作

阅读

阅读部分共有5个题型

Multiple-choice, choose single answer

  • 给出一段文本
  • 选出正确答案
  • 单选

评分标准

  • 选对1个1分

交叉评分

  • 阅读

Multiple-choice, choose multiple answer

  • 多选题
  • 不确定就当单选

评分标准

  • 选对1题1分，选多会倒扣一分

交叉评分

  • 阅读

Re-order paragraphs

  • 排序题
  • 打乱顺序
  • 不一定连贯
  • 根据逻辑性排序

评分标准

  • 选对1个1分

交叉评分

  • 阅读

Fill in the blanks（单词库）

  • 类似完形填空
  • 给出单词库进行选择
  • 一个单词只选一次

评分标准

  • 选对1个1分

交叉评分

  • 阅读

Fill in the blanks（四选一）

  • 类似完形填空
  • 一个空给出4个单词

评分标准

  • 选对1题1分

交叉评分

  • 阅读
  • 写作

考完阅读后中间会有十分钟休息时间

听力

听力部分共有8个题型

Summarize Spoken Text

  • 播放一段录音
  • 可能有背景音
  • 口音多样
  • 做总结
  • 听到关键词或关键句后做成笔记
  • 50-70个单词
  • 单独计时间

评分标准

  • 内容2分
  • 形式2分
  • 语法2分
  • 词汇2分
  • 拼写2分

交叉评分

  • 听力
  • 写作

Multiple-choice, choose multiple answer

  • 多选题
  • 不确定就当单选

评分标准

  • 1个1分
  • 选多会倒扣1分

交叉评分

  • 听力

Fill in the blanks（单词库）

  • 类似完形填空
  • 挖掉几个空
  • 给出一段文本
  • 播放录音

评分标准

  • 1空1分

交叉评分

  • 听力
  • 写作

Highlight correct summary

  • 播放录音
  • 四选一

评分标准

  • 1个1分

交叉评分

  • 听力
  • 阅读

Multiple-choice, choose single answer

  • 单选题

评分标准

  • 1个1分

交叉评分

  • 听力

Select missing word

  • 放一段录音
  • 选项前播放bi声
  • 选择正确单词

评分标准

  • 1个1分

交叉评分

  • 听力

Highlight incorrect words

  • 给出一段文本
  • 播放一段录音
  • 文本与录音中某些单词有差别
  • 选出不一样的单词
  • 没听出来就不要选了

评分标准

  • 选对1空1分，选错1空倒扣1分

交叉评分

  • 听力
  • 阅读

Write from dictation

  • 播放一句录音
  • 写出听到的句子

评分标准

  • 1个单词1分

交叉评分

  • 听力
  • 写作

Win10重装指北

发表于 2020-07-13 分类于 Tech 阅读次数：
本文字数： 5.1k 阅读时长 ≈ 5 分钟
写一个Win10重装教程，另外偷个懒，把office的安装教程也合进来了hhh

必备素材

  • 容量≥8G的U盘一个
  • 能正常使用的电脑一台
  • Refus启动盘制作工具
    • 下载地址
    • 在网页下面的Download中选择Rufus x.xx Portable
  • Win10镜像
    • 1909及以前版本
      • 下载地址
      • 点击左边栏的操作系统
      • 选择想要安装的版本
      • 以Windows 10, Version 1909 (updated Jan 2020)为例
        • 1909表示这是2019年9月大更新的版本（下一个版本号为2004，意思就是2020年4月大更新的版本）
        • updated Jan 2020表示这个镜像包含了2020年1月份的小更新
      • 可以看到有4个选项，以Windows 10 (business editions), version 1909 (updated Jan 2020) (x64) - DVD (Chinese-Simplified)为例
        • business editions是我们需要的版本，包含企业版、专业版、教育版、专业工作站等版本
        • x64代表64位系统（32位系统为x86），如果没有特殊需要建议安装64位
      • 点击详细信息，可以看到一个ed2k开头的链接
      • 复制这个链接，用迅雷下载镜像文件
    • 2004及之后的版本
      • 下载地址
      • 使用邮件或者第三方账户注册并登录
      • 有能力的朋友可以捐助一下，网站作者已经坚持免费12年了
      • 选择Windows10并点击详细信息
      • 以Windows 10 (business edition), Version 2004 (Updated June 2020) (x64) - DVD (Chinese-Simplified)为例
      • 点击后面的复制按钮
      • 打开相应的下载软件（这里推Motrix）并下载镜像

制作安装盘

  • 插入U盘
  • 以管理员身份运行Refus
  • 在设备中选择你的U盘
  • 引导类型选择前一栏保持默认的镜像文件（请选择）然后点击后面的选择按钮
  • 找到下载的Win10镜像并打开
  • 点击最下面的开始
  • 等待制作完成

BIOS修改启动项

下面的选项表述不一定准确，以电脑BIOS标注为准，意思对即可

  • 重启电脑
  • 开机的时候狂按F2/F10/Delete（不同的电脑进入BIOS的按键不一样）
  • 进入BIOS
  • 找到启动顺序/启动项管理
  • 关闭安全启动/Secure Boot
  • 新电脑
    • 引导方式选择UEFI Only
    • 将你的U盘设置为第一启动项（部分电脑需要使用F5/F6移动到最上方）
    • 如果有两个U盘选项，例如UEFI:USBXXX和USBXXX，选择带有UEFI的那一个
    • 保存设置并退出BIOS
  • 老电脑（不支持UEFI启动的）
    • 将你的U盘设置为第一启动项（部分电脑需要使用F5/F6移动到最上方）
    • 保存设置并退出BIOS
  • 等待电脑重启

转换硬盘分区格式

这一步只针对UEFI启动，非UEFI的请跳过

  • 等电脑转完圈后会进入语言选择页面
  • 使用Shift+F10打开终端
  • 输入diskpart打开磁盘管理
  • 输入list disk，列出硬盘，会显示硬盘的序号（0/1/2等）
  • 找到安装目标磁盘的序号，输入select disk 序号回车
    • 注意不要选到自己的U盘
  • 输入clean回车
  • 输入convert gpt转换gpt格式
  • 再次输入list disk，观察GPT下面是否出现星号
  • 有星号的话就可以关闭终端了

安装设置

不同版本的Win10安装过程稍有不同，以下步骤不会在所有系统中出现

  • 选择安装语言后点击下一步
  • 点击现在安装
  • 进入激活Windows页面，选择我没有产品密钥
  • 进入选择要安装的操作系统页面，根据需要选择要安装的版本并点击下一步
  • 进入适用的声明和许可条款页面，勾选我接受许可条款并点击下一步
  • 进入你想执行哪种类型的安装页面，选择自定义：仅安装Windows（高级）
  • 进入你想将Windows安装在哪里页面，根据驱动器的大小判断硬盘，如果有固态硬盘的话选择固态硬盘
  • 点击右下方的新建，此时会提示创建其他分区，点击确定
  • 分区创建完成后点击下一步
  • 等待系统安装完成，这一步耗时相对较长
  • 进入提示Windows需要重启才能继续的页面，会有10秒的重启倒计时，此时拔出U盘并点击立即重启
  • 重启后会提示让我们先从区域设置开始，选择中国并点击是
  • 提示这种键盘布局是否合适，选择需要的键盘布局并点击是
  • 有需要的话可以添加第二种键盘布局，步骤同上，没有需要点击跳过
  • 连接网络，不想连接或者没有网络的话点击左下角的我没有Internet连接
    • 不连接的话会有第二遍提示：当你连接到Internet时，有更多要发现的内容，点击左下角的继续执行有限设置
  • 输入用户名和密码并确认密码
  • 三个安全问题胡乱写就行，从来没有用到过这个东西
  • 进入在具有活动历史记录的设备上执行更多的操作页面，如果你有多台电脑的话可以选择是，开启后可以同步不同电脑上的活动记录，不过个人感觉没啥卵用
  • 进入从数字助理获取帮助页面，选择接受就会开启Cortana，其实也没什么卵用
  • 进入为你的设备选择隐私设置页面，根据需要进行勾选，我个人只开启了位置与查找我的设备两个选项
  • 点击接受后就设置完成了，等待一会就会进入桌面

更新

进入桌面后建议联网，然后进入设置-更新和安全-检查更新，等待所有更新安装完成后点击下面出现的立即重启，这个过程耗时较长，取决于网速。另外Win10会自动在后台安装驱动，如果当前屏幕分辨率很怪的话不用慌，过一会安装完驱动就好了。

Office安装与激活

安装方法如下

  • 下载Office Tool Plus，建议下载完整版
  • 解压文件并使用管理员身份运行Office Tool Plus.exe
  • 点击部署
  • 在产品中勾选需要的版本
    • 一般选择的是Office 专业增强版 2019 批量版 ProPlus2019Volume
  • 在应用程序中勾选需要的软件
  • 在右侧的体系结构中选择x64，32位系统请选择x86
  • 选择相应的通道
    • 一般选择每月通道
    • 企业版Win10可能会提示不支持通道，可以换成Office 2019 企业长期版
    • 或者先选择测试通道或者开发通道后，再重新选择每月通道，这样就不会报错了
  • 安装方式保持默认的在线安装
  • 安装模块保持默认的Office部署工具
  • 勾选安装完成后创建桌面快捷方式
  • 点击右下角的开始部署

Office同样可以使用KMS激活,步骤如下

  • 使用管理员身份运行Office Tool Plus.exe
  • 点击激活
  • 在KMS管理中输入KMS服务器地址kms.03k.org并点击应用服务器地址
  • 在密钥管理中输入相应版本的秘钥并点击安装密钥
  • 点击右下角的激活

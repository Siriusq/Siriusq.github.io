Windows 11 右键菜单改回 Windows 10 样式

发表于 2022-05-26 分类于 折腾 阅读次数： 本文字数： 337 阅读时长 ≈ 1 分钟
巨硬迷惑操作之两级右键菜单，真该把提出这个需求的产品经理祭天，好在还能通过祖传注册表修改大法改回去

前排提示，修改注册表有风险，请谨慎操作

命令行修改

下面是命令行大法，按下WIN+R，输入cmd后回车，运行下面的指令

  • /f表示不弹出确认窗口，静默执行添加注册表任务
  • /ve表示添加到注册表中的项为空值null

按下CTRL+SHIFT+ESC打开任务管理器，找到Windows资源管理器，点击右下角重新启动就完事了

注册表编辑器手动修改

按下WIN+R，输入regedit后回车，打开注册表编辑器

定位到HKEY_CURRENT_USER\SOFTWARE\CLASSES\CLSID，右键单击CLSID，选择新建-项，命名为{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}

右键单击新建的项，选择新建-项，命名为InprocServer32

双击打开InprocServer32右侧的默认，不需要填写任何东西，直接点击确定，然后关闭注册表编辑器，并重启Windows资源管理器

然后就能见到亲切的经典Window10右键菜单咯

软件整理

整理收录了一些我常用/用过的软件、网站以及浏览器扩展，以开源/免费软件为主。

基本软件

浏览器

  • Vivadli 🪟🍎 Chromium内核，支持垂直标签页
  • Firefox 🔓🪟🍎
  • Zen 🔓🪟🍎 基于Firefox，支持垂直标签页

输入法

  • Rime 🔓🪟🍎 中文输入法，需要自己配置，建议搭配雾凇拼音或薄荷拼音快速配置
  • 哈利路亚 🔓🪟🍎 支持自动补全、拼写校正与即时翻译功能的英文输入法

看图

  • nomacs 🔓🪟 支持格式丰富，可转换图片格式
  • FlyPhotos 🔓🪟 UI美观，仅支持看图
  • Honeyview ⛓️‍💥🪟 支持格式丰富，可转换图片格式，配合Bandzip可以预览压缩包内图片

截图

  • ShortX 🔓🪟 功能丰富，自定义程度高，支持OCR
  • Snipaste 💸🪟🍎 有MS商店版本

屏幕录制

  • OBS Studio 🔓🪟🍎

视频播放

  • mpc-hc 🔓🪟
  • mpc-be 🔓🪟
  • VLC 🔓🪟🍎
  • K-Lite Codec Pack Full 🪟 带有MPC-HC、madVR以及LAV的视频编解码器软件包，简单设置参考基于 MPC-HC 和 madVR 的播放器配置入门 | VCB-Studio
  • Potplayer 🪟 功能最多，但是有黑历史

压缩解压

  • Peazip 🔓🪟🍎 双击解压后可打开自动解压文件夹
  • 7-Zip 🔓🪟 体积小，性能强
  • Bandzip 双击解压后可自动打开解压文件夹并删除压缩包
    • v6 ⛓️‍💥🪟 需关闭自动更新使用
    • v7 💸🪟🍎 有广告
  • Keka 🔓🍎 解压缩

文件下载

  • FDM 🪟🍎 可搭配浏览器扩展使用
  • Motrix ⛓️‍💥🔓🪟🍎 UI美观
  • imFile 🔓🪟🍎 Motrix 的继任者

安全/密码

  • ESET 💲🪟🍎 系统资源占用不多，HIPS功能强大，推荐Internet Security版本
  • 火绒安全 🪟
  • KeepassXC 🔓🪟🍎 密码管理工具，可以通过OneDrive同步数据库，iOS端可以用StrongBox/KeePassium
  • KnockKnock 🔓🍎 软件行为监控
  • LuLu 🔓🍎 防火墙

创意设计/生产力

  • GIMP 🔓🪟🍎
  • Paint.NET 🪟
  • Photoshop 💲🪟🍎
  • webp-converter 🔓🪟 将图片批量转换为webp格式

视频剪辑

  • Kdenlive 🔓🪟🍎

  • Premiere Pro 💲🪟🍎

音频剪辑

  • SketchUp 💸🪟🍎 基础网页版免费使用

Office

  • Kindle for PC 🪟🍎 内置词典

翻译

  • DeepL 💸🛜
  • 欧陆词典 💸🪟🍎

绘图

  • Excalidraw 🔓🛜💸 手绘风格

远程会议

  • Microsoft Teams 💸🪟🍎

  • Sci-Hub 🛜 免费下载文献，当代雷锋
  • BibGuru 🛜 引用格式整理

效率/工具

快速启动

  • Flow.Launcher 🔓🪟
  • PowerToys Run 🔓🪟
  • utools 💸🪟🍎 功能最多
  • rubick 🔓🪟🍎 高仿utools

远程控制

  • Parsec 💸🪟🍎 局域网内免费，Windows PC 远程 mac，画质高但是占用资源也高
  • 向日葵 💸🪟🍎 有广告弹窗
  • VNC Viewer 🪟🍎
  • Mouse Without Borders 🪟 局域网内控制

网络工具

  • Clash Verge Rev 🔓🪟🍎 科学上网
  • V2RayN 🔓🪟🍎 科学上网
  • Wireshark 🪟🍎 抓包工具

鼠标增强

  • WGestures 💸🪟🍎 鼠标手势控制
  • SnipDo 🪟 悬浮文本工具，当选中文字时会出现悬浮框，可以快捷执行复制粘贴等操作，也可以自定义自动化操作
  • MOS 🔓⛓️‍💥🍎 模拟平滑滚动，消除鼠标滚轮的卡顿感

文件管理/传输

  • Everything 🪟 快速搜索文件
  • QuickLook 🔓⛓️‍💥🪟 空格键快速预览文件
  • LocalSend 🔓🪟🍎 局域网文件传输共享
  • Apple设备 🪟 iOS 设备与 Windows 间传送文件
  • Onedrive 🪟🍎
  • SpaceSniffer 🪟 将每个文件占用的磁盘空间可视化
  • Duplicate Cleaner 💲🪟 找出并删除重复文件
  • File Converter 🔓🪟 右键菜单快速本地文件转换

桌面美化

  • Rainmeter 🔓🪟
  • Wallpaper Engine 💲🪟 动态壁纸
  • TranslucentTB 🔓🪟 透明任务栏
  • MacType 🔓🪟 优化Windows的字体渲染
  • Context Menu Manager 🔓🪟 管理右键菜单
  • Windhawk 🔓🪟 调整任务栏与开始菜单

系统工具

  • Microsoft PowerToys 🔓🪟 Windows 系统优化与功能扩展
  • Traffic Monitor 🔓⛓️‍💥🪟 悬浮窗监控系统状态
  • RunCat_for_windows 🔓🪟 用猫猫动画在任务栏展示CPU占用率
  • InputTip 🔓🪟 输入法状态(中文/英文/大写锁定)提示工具
  • DISM++ 🔓⛓️‍💥🪟 系统优化工具箱
  • AIDA64 💲🪟 详细查看系统的各项参数并进行测试
  • Rufus 🔓🪟 系统安装盘制作工具
  • DDU 🪟 显卡驱动完整卸载
  • 图吧工具箱 🪟 各种系统测试工具的合集
  • iGlance 🔓⛓️‍💥🍎 在菜单栏显示系统信息，如CPU温度、内存使用以及网速
  • CheatSheet 🍎 长按command键查看当前软件的快捷键
  • Rectangle 🔓🍎 分屏软件

虚拟机

  • VMWare 🪟🍎 被博通收购后免费了
  • VirtualBox 🔓🪟🍎

硬件控制

  • EarTrumpet 🔓🪟 独立控制不同软件的音量大小
  • Twinkle Tray 🔓🪟 显示器亮度调节
  • BetterDisplay 💸🍎 在2K外接显示器上开启HiDPI
  • AlDente 💸🍎 设置电池充电电量上限，延长电池寿命
  • AI Suite3 🪟 华硕主板风扇转速控制
  • Logitech G Hub 🪟🍎 罗技鼠标设置
  • Trident Z 控制台 🪟 芝奇幻光戟灯效控制
  • OpenRGB 🔓🪟
  • Armoury Crate 🪟 华硕主板的控制软件，界面很炫酷，但是Bug太多，依托答辩
  • Aura Sync 🪟 华硕神光同步灯控软件，功能单一，但bug少多了

驱动

  • AMD rx6500xt 显卡驱动 🪟

  • Dell Display Manager 以及校色文件 🪟

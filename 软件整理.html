软件整理

发表于 2019-10-03 更新于 2024-09-12 分类于 Windows 本文字数： 4.3k 阅读时长 ≈ 16 分钟
记录下平常用的软件和官网的链接，免得重装系统之后还得重新一个一个找2333333

又加上了一些好用的网站、浏览器插件以及暴力猴油猴脚本的链接

常用软件

也许可以叫装机必备系列？（其实是不知道算哪一类软件）

图片查看

Honeyview

又名为蜂蜜浏览器，支持PSD文件预览，快捷查看EXIF信息，以及批量编辑图片格式

解压缩

Bandzip

功能强大，UI美观，可以设置双击解压缩后自动打开文件夹并删除原始压缩文件。需注意7.0及以后的版本有广告以及付费解锁功能，介意请下载6.29版本使用并关闭自动更新

7-Zip

免费开源无广告的解压软件，就是UI比较简陋

下载器

FDM

全称 Free Download Manager，可以安装插件配合Chrome使用

Motrix

颜值很高的开源下载软件

视频播放

Potplayer

颜值高，功能也很强大，不过官网经常抽风。需注意20年的版本出现了弹窗广告，请下载2019年的老版本并关闭自动更新，或者安装后使用防火墙禁止其联网

mpv

免费开源的视频播放器，但是设置没有GUI，上手难度较大

vlc

免费开源的视频播放器，与mpv相比功能少了些，但是设置有GUI

浏览器

Microsoft Edge

新Edge换用了Chromium内核，比原来的老浏览器香多了，而且登录微软账号后不用梯子就可以同步，微软自己的扩展商店同样不需要梯子，chrome瞬间就不香了。Win11系统已预装。

Google Chrome

建议配合谷歌账号食用，安装后登陆谷歌账号就可以同步安装原来装的所有的插件和皮肤

Mozilla Firefox

火狐浏览器，也很好用（虽然是备胎）

办公写作

Onenote

微软出品的笔记本，支持云同步，windows系统预装了uwp版本，安装office的时候也可以选择再安装一个桌面版本，桌面版本功能更多一些

Microsoft Office

Office基本是必装软件了，现在新买的品牌机或者笔记本会预装office365。自己组装的台式机建议使用 Office Tool Plus 安装

TeX Live LATEX

论文排版软件，对数学公式的支持很强大

XMind

思维导图

Grammarly

语法错误提醒纠正，救万千留学生于水火的神器，支持嵌入Word运行

电子书阅读

Aquile Reader

UWP软件，epub电子书阅读器

Book Bazaar Reader

UWP软件，支持EPUB，MOBI，PDF, FB2，TXT格式

PDF编辑

CleverPDF

一款PDF工具箱，不想下载的话可以直接用在线工具，下载的话免费版就够用了，体积很小

Nitro PDF Pro

功能很齐全，但是没有中文版，付费软件

Adobe Acrobat Pro DC

最强的PDF软件，但是价格很贵

视频会议

ZOOM

单纯的视频会议客户端，国外用的比较多

Microsoft Teams

功能很多，算是办公协作平台了，国外用的比较多

文献管理

Zotero

文献管理软件，可以搭配插件使用

ZotFile

插件，可以重命名下载的PDF文献，同步文献到OneDrive等网盘

Better BibTeX for Zotero

插件，自动生成BibTeX文件

实用工具

文件传输

Onedrive

微软出品的网盘，windows系统预装，多端同步不限速。免费版有5G空间，可以淘宝花1块钱扩容到15G

LocalSend

LocalSend 是一个跨平台应用程序，它使用 REST API 和 HTTPS 加密实现设备之间的安全通信，不需要互联网连接或第三方服务器。

iTunes

iPhone与Windows间传输文件

文件快速预览

按下空格键预览文件，模仿macOS的功能

QuickLook

UWP版本，可安装扩展插件

Seer

有UWP版本

快捷启动

类似于Macos的spotlight，可以配合Everything使用

uTools

UI美观，插件非常丰富

Wox

原来用的，已经很长时间没有更新了

屏幕录制

OBS Studio

免费开源的录屏软件，功能很强大，有桌面版和UWP版

鼠标功能增强

WGestures

鼠标手势

Logitech G Hub

罗技鼠标的侧键自定义软件

音量调节

EarTrumpet

UWP软件，可以分别设置不同应用的音量大小，避免耳聋（滑稽）

屏幕截图

Snipaste

按F1就可以截图，还可以在图片上添加注释，有UWP版本

文件快速搜索

Everything

文件快速搜索，结果秒出，比系统自带的快很多

系统增强优化

Microsoft PowerToys

微软官方出品的系统增强软件，有很多实用好玩的功能

Dism++

有很多优化系统的小功能，如清理垃圾备份系统

TrafficMonitor

开源系统信息监测工具，包括网速、CPU占用等，支持悬浮窗与任务栏两种显示模式

安全软件

ESET

来自斯洛伐克的付费安全软件，占用系统资源不多，HIPS功能很强大，我使用的是Internet Security版本，也是用的相对较多的版本

KeePassXC

PC端开源的密码管理工具，可以通过OneDrive同步数据库，iOS客户端推荐用StrongBox。不建议用浏览器存储密码，毕竟浏览器密码管理工具就是在裸奔（手动狗头）

文件清理

Duplicate Cleaner

付费软件，能找出硬盘里的重复文件并快速删除

SpaceSniffer

可以将每个文件占用的磁盘空间可视化，快速找到占用空间最大的文件

虚拟机

VMWare Workstation Pro

支持运行多种系统（因为版权原因，运行macOS需要打补丁）。Windows下不放心的软件就扔到里面跑

夜神模拟器

安卓模拟器

系统重装

Refus

用来刻录系统安装盘，轻量化很干净

UltraISO

光盘刻录软件，但最广泛的用途是重装系统

远程控制

VNC Viewer

远程控制软件，多平台支持

向日葵

国产的远程控制软件，新版本有弹窗，需要绑定手机号，只推荐帮国内朋友远程解决问题使用

Mouse Without Borders

微软车库项目，仅支持局域网内控制，仅限Windows系统。另：已加入PowerToys豪华套餐，名为无边框的鼠标（好浓的机翻味道）

桌面美化

Rainmeter

有很多主题与插件，好好折腾的话蛮好看的，然鹅我主要用来看电脑的硬件情况

Wallpaper Engine

Steam上的付费软件，可以从创意工坊下载各种动态壁纸

TranslucentTB

UWP软件，可以自定义任务栏的背景样式,支持透明背景，但是windows更新后容易失效

硬件相关

硬件信息查看

AIDA64

付费软件，可免费试用，可以详细查看系统的各项参数并进行各种测试，超频的时候比较有用

图吧工具箱

图吧出品，集成了很多好用的软件，包括跑分、硬件信息查看等

华硕灯效控制

Armoury Crate

华硕主板的控制软件，界面很炫酷，但是越更新越臃肿，Bug也变多了

G.SKILL Trident Z Lighting Control

芝奇灯条的控制软件，控制幻光戟，皇家戟，焰光戟灯效，在Armoury Crate不能正常控制内存灯效的时候用

Aura Sync

华硕神光同步灯控软件，功能单一，但bug少多了

风扇调速

AI Suite

华硕主板管理软件，我主要用来调节风扇转速

显示器调节

Dell Display Manager

戴尔显示器控制，支持快捷键切换视频输入接口

Dell P2418D驱动

适用于戴尔P2418D的显示器驱动，包含icc校色文件

Twinkle Tray

UWP软件，可以像调节音量一样调节显示器亮度，不用再吧嗒吧嗒的用OSD了

程序员脱发合集

会让程序员脱发的软件，用霸王都挡不住

IDE

Visual Studio

windows平台最稳定的IDE，不会死机不会卡🐶

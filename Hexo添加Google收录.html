Hexo添加Google收录

发表于 2019-04-16 更新于 2022-06-12 分类于 网站配置 阅读次数： 本文字数： 275 阅读时长 ≈ 1 分钟

目前网站还不能够通过搜索引擎搜索到，想要被搜到的话需要添加搜索引擎收录，这里以我添加的Google为例。
步骤包括：

  • 生成站点地图
  • 验证站点

生成站点地图

  1. 在博客根目录下打开Git bash
  2. 执行命令npm install hexo-generator-sitemap --save安装站点地图生成插件
  3. 在博客根目录下的配置文件_config.yml中添加下列内容
  4. 执行hexo g即可在博客根目录/public中生成sitemap.xml文件

验证站点

  1. 打开Google网站站长,点击右上角登陆
  2. 选择网址前缀并输入站点域名，点击继续
  3. 下载Google的验证文件并存放在博客根目录/themes/next/source中
  4. 执行hexo cl,hexo g,hexo d三连
  5. 回到Google网站站长进行验证
  6. 在Google中搜索site：域名就可以看到自己的网站了

现在已经可以在Google上搜索到自己的文章和网站了，填坑结束~~~
